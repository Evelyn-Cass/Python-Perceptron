# 🧠 Python-Perceptron

A Python project for implementing the Perceptron algorithm, built with Jupyter Notebook. Demonstrates the fundamentals of neural networks and binary classification.

## 📱 About The Project

Python-Perceptron is a simple and practical implementation of the Perceptron algorithm, one of the oldest and most fundamental neural network models. The project uses multiple datasets to train and test the perceptron for binary classification tasks.

This project was created for study purposes and to demonstrate the Perceptron algorithm using Python.

## ✨ Features

- Perceptron algorithm implementation from scratch
- Binary classification
- Multiple training and testing datasets
